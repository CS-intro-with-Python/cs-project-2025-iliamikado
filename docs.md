# API routes documentation

## /hello (GET)
- returns "Hello, World!"

## / (GET)
- shows the game page

## /board (GET)
- returns an array of all opened cells

## /click (POST)
- opens a cell
- returns newly opened cells

## /reset (POST)
- restarts the game

## /login (GET)
- return page for register or login

## /register (POST)
- register new user

## /login (POST)
- login as user
